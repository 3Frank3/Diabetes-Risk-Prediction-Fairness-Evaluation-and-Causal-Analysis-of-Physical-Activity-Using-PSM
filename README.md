# Effect-of-Physical-Activity-on-Diabetes-A-Propensity-Score-Matching-and-Machine-Learning-Analysis

This repository contains the code and data for the project "Effect of Physical Activity on Diabetes: A Propensity Score Matching and Machine Learning Analysis". The project aims to investigate the impact of physical activity on diabetes risk using propensity score matching and machine learning techniques.

## Data 
The dataset used in this project is sourced from the National Health and Nutrition Examination Survey (NHANES). It includes various demographic, lifestyle, and health-related variables. The data has been preprocessed to handle missing values and ensure quality for analysis.

## Code
The code is organized into Jupyter notebooks, which include:
- `PSM.ipynb`: This notebook contains the code for performing propensity score matching to balance the covariates between the treatment and control groups.
- `Machine_Learning_Analysis.ipynb`: This notebook includes the code for applying machine learning algorithms to predict diabetes risk based on the matched dataset.    
- `Data_Preprocessing.ipynb`: This notebook details the steps taken to preprocess the data, including handling missing values and encoding categorical variables.

## Results
The results of the analysis are presented in the notebooks, including tables and visualizations that illustrate the findings. The propensity score matching effectively reduced confounding, and the machine learning analysis provided insights into the predictors of diabetes risk.

## Conclusion
The project demonstrates the use of propensity score matching to control for confounding variables and the application of machine learning techniques to analyze the relationship between physical activity and diabetes risk. The findings suggest that physical activity has a significant impact on reducing the risk of diabetes, and the machine learning models can help identify key predictors of diabetes risk.
